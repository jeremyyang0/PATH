# 测试方法树

测试方法树用于展示项目中 `method` 目录下的所有测试方法，支持中文包名映射、搜索和跳转。

## 功能概述

- 按照 Python 包结构展示 `method` 目录
- 自动将英文包名替换为中文（从 `__init__.py` 读取）
- 展示每个模块中的方法列表及文档字符串
- 支持搜索、跳转、拖拽操作

## 解析机制

### 扫描范围

插件会扫描项目中 `method` 目录下的所有 Python 文件，解析：

- 包结构（目录层级）
- 方法定义（`def` 关键字）
- 方法文档字符串（docstring）

### 中文包名映射

插件会从各包的 `__init__.py` 文件中提取中文名称，支持以下方式：

**类文档字符串**

```python
class LogicEditor:
    """逻辑编辑器模块"""
    pass
```

### 映射示例

原始路径：
```
logic
└── logic_editor
    └── dialog
```

映射后显示：
```
逻辑
└── 逻辑编辑器
    └── 对话框
```

## 操作说明

### 浏览方法

- **展开/收起**：点击文件夹图标
- **查看方法文档**：鼠标悬停在方法节点上

### 搜索方法

1. 点击搜索按钮
2. 输入关键词（方法名、文档字符串、文件路径）
3. 实时显示匹配结果
4. 搜索时自动展开匹配的节点

### 双击跳转

| 节点类型 | 双击效果 |
|----------|----------|
| 文件夹节点 | 打开该包的 `__init__.py` 文件 |
| 方法节点 | 跳转到方法定义位置 |

### 拖拽方法

1. 找到目标方法
2. 拖拽到编辑器
3. 自动生成方法调用路径

## 方法识别

### 普通方法

```python
def login(username: str, password: str):
    """用户登录"""
    pass
```

显示为：`login - 用户登录`

### 断言方法

如果方法名包含 `assert`，会被标记为断言方法：

```python
def assert_login_success():
    """断言登录成功"""
    pass
```

## 状态保持

与元素树相同，方法树也会保存：

- 展开/收起的节点
- 搜索关键词

## 刷新视图

当 `method` 目录下的文件发生变化时（即文件保存时），会自动更新方法树相关内容。

## 常见问题

### 中文包名没有显示

- 检查 `__init__.py` 文件是否存在
- 确认文件中包含中文类名或属性
- 确保文件编码为 UTF-8
- 刷新视图

### 方法没有显示

- 检查方法定义是否符合 Python 语法
- 确认文件在 `method` 目录下
- 刷新视图
