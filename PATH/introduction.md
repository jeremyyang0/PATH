# PATH 插件简介

> **P**lugin **A**uto **T**est **H**omie - Python 自动化测试助手

## PATH是什么？

**PATH** 是为派兹自动化测试开发者设计一款 VSCode 扩展插件。插件智能解析用例工程中封装的**元素**和**操作方法**，提供直观的树形结构浏览界面，**深度集成禅道（Zentao）**和 **AI 辅助**功能，为编写测试用例提供了简单便捷的开发体验，让测试工程师可以专注于业务逻辑的实现，大幅提高了测试效率和准确率。🚀

## 核心价值

- **智能解析**：自动识别和组织测试元素及方法，无需手动维护文档
- **可视化浏览**：直观的树形结构，快速定位和导航代码
- **禅道集成**：一键同步测试用例，自动生成标准化测试脚本
- **AI 辅助**：智能生成测试代码，大幅降低重复性工作
- **拖拽编辑**：可视化操作，快速构建测试流程
- **调试支持**：自动配置调试环境，支持右键快速调试
