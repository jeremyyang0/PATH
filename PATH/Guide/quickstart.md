# 快速开始

## 第一步：配置插件

打开 VSCode 设置（`Ctrl+,`），搜索 `path`，根据需要配置以下信息：

### 禅道配置（可选）

| 配置项 | 说明 | 示例 |
|--------|------|------|
| path.zentao.host | 禅道服务器地址 | `10.0.18.30` |
| path.zentao.username | 禅道用户名 | `automation` |
| path.zentao.password | 禅道密码 | - |

### 应用配置（可选）

| 配置项 | 说明 | 示例 |
|--------|------|------|
| path.appName | 默认应用名称，配置后创建用例时自动填充 | `Logic` |

### AI 配置（可选）

| 配置项 | 说明 | 示例 |
|--------|------|------|
| path.ai.url | AI 服务地址 | `http://127.0.0.1:8045/v1` |
| path.ai.apiKey | API 密钥 | - |
| path.ai.model | 模型名称 | `gemini-3-flash` |

## 第二步：打开树形视图

点击 VSCode 左侧活动栏的 **PATH** 图标，展开两个树形视图：

- **测试元素树**：显示所有 Ele 变量
- **测试方法树**：显示所有测试方法

## 第三步：基本操作

### 浏览代码

1. 在树形视图中展开节点
2. 单击元素节点，跳转到源码定义
3. 使用搜索框快速定位

### 创建测试用例

1. 在资源管理器中右键点击目标文件夹
2. 选择 **Create Case**
3. 输入禅道 ID（可选）
4. 输入应用名称
5. 自动生成测试文件

### 编写测试代码

**方式一：拖拽操作**
1. 在元素树中找到目标元素
2. 拖拽到编辑器
3. 自动生成变量路径

**方式二：右键操作**
1. 在元素树中右键点击元素
2. 选择操作类型（点击、双击）
3. 自动生成操作代码

**方式三：AI 生成**
1. 编写步骤注释
2. 右键选择 **AI Generation**
3. 自动匹配方法并生成代码

## 常用快捷操作

| 操作 | 说明 |
|------|------|
| 单击元素节点 | 跳转到源码定义 |
| 双击方法节点 | 跳转到方法定义 |
| 双击文件夹节点 | 打开 `__init__.py` |
| 拖拽元素 | 生成变量路径 |
| 刷新按钮 | 重新扫描项目 |
| 搜索按钮 | 过滤树形视图 |
