# 安装指南

## 从 VSIX 安装

1. 下载最新的 `.vsix` 文件（如 `path-plugin-auto-test-homie-1.4.9.vsix`）
2. 打开 VSCode
3. 按 `Ctrl+Shift+P` 打开命令面板
4. 输入 `Install from VSIX`
5. 选择下载的 `.vsix` 文件
6. 等待安装完成，重启 VSCode


## 验证安装

安装成功后，VSCode 左侧活动栏会出现 **PATH** 图标。点击图标可以看到：

- 测试元素树
- 测试方法树

## 卸载

1. 打开 VSCode 扩展面板（`Ctrl+Shift+X`）
2. 搜索 `PATH`
3. 点击卸载按钮
